# vi

|Command|Description|
|-|-|
|/sth| Will search for sth in text |
|i| Switch to insert mode |
| [Esc] | Escape insert mode |
| :wq | Write and quit |
| q! | Force quit |
|[Esc] d d| Remove line|
| [Alt] u | undo changes (Ctrl+Z) |
| [Esc] :%s/origin/changed/ | replace all "origin" with "changed" in text
